# Argus Frontend

The analyst dashboard — **React + TypeScript + Vite + Tailwind**, with Chart.js + D3 for visualization.

> Scaffolded at **Milestone M3**. Planned views: live alert feed (WebSocket), time-series of attack frequency, a D3 world attack map, and an alert triage workflow (acknowledge / resolve / mark false-positive), with role-gated access.

To scaffold when we reach M3:

```bash
npm create vite@latest . -- --template react-ts
npm install
npm install -D tailwindcss postcss autoprefixer
npm install chart.js react-chartjs-2 d3
```
