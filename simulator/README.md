# Argus Attack Simulator

A replay/simulation harness that generates synthetic attack traffic — DoS bursts, brute-force login storms, port scans, and SQL-injection probes — and feeds it into the Argus ingest API so detection can be demonstrated live and measured (precision / recall, false-positive rate).

> Built at **Milestone M6**. This directly fulfills the original report's "simulated intrusion scenarios to evaluate system responsiveness, accuracy, and alert quality" goal.
